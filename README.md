### O que eu fiz nesse projeto? 

  - Com ele, eu pratiquei uma das Design patterns, a *Strategy Design Pattern*, inspirado em
  uma das aulas do meu curso [C# Essencial feito pelo Macoratti](https://www.udemy.com/course/curso-c-essencial-2023-bonus-linq/). Consegui criar 
  um projetinho básico e pequeno, um sistema para calcular fretes que varia de acordo com o tipo de entrega, somente para aplicar o que tinha aprendido e internalizar.

  - E, para a parte técnica, os conceitos e técnicas que utilizei foram os *princípios SOLID*, através do uso de interfaces, e organização dos arquivos de código. 
  Além de alguns recursos mais atuais da linguagem C# para melhorar a qualidade do código e evitar muito trabalho.
